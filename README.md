# Padel Mix & Match

A simple, mobile-first web app to run a 3-hour Padel **Mix & Match** session.

- 4–16 players, up to 3 courts
- Random teams every round
- Simplified scoring: each game totals 24 points (e.g. `17:7`, `12:12`)
- Pause / resume players who take a break
- Live total-points ranking
- Works **offline** once loaded (PWA), installs to your home screen
- No accounts, no backend, no cost — all state lives on the host's phone
